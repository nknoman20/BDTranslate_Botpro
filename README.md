# BD Translate Bot 🇧🇩

A simple Telegram bot for real-time message translation using LibreTranslate and Flask.

## Features
- 🌐 Auto-detects source language
- 🈯️ Supports 40+ languages
- 🧠 Remembers user language choice
- 📊 /stats to view your usage
